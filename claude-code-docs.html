Claude Code 문서

AI 기반 코딩 도구의 완벽한 가이드

Claude Code는 터미널에서 작동하는 Anthropic의 AI 코딩 도구입니다.

자연어 명령을 통해 코드베이스를 이해하고, 루틴 작업을 실행하며, 복잡한 코드를 설명하고, Git 워크플로우를 처리합니다.
